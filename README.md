# AI Academic Strategist - UI Updates Version

This is the **UI/UX enhanced version** of the AI Academic Strategist project.

## What's Different

This branch contains updated UI/UX improvements while maintaining the core functionality.

## Getting Started

Run the development server on a **different port** (3001):

```bash
npm run dev:ui
# or
npm run dev -- -p 3001
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

## Features

- Enhanced user interface
- Improved user experience
- New components and layouts

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Supabase

## Branches

- `main` - Production version (localhost:3000)
- `ui-updates` - This version (localhost:3001)

## Notes

This runs on **port 3001** to avoid conflicts with the main branch running on port 3000.
