## 🌐 Live Demo

## 🚀 Vision

Students don't fail because information is unavailable. They fail because information is overwhelming, unstructured, and poorly prioritized.

We are building an AI Academic Strategist — a system that doesn't just answer questions, but understands academic context, guides learning decisions, and optimizes exam preparation.

---

## ✨ Features

### 📚 Study Mode
- Doubt solving (syllabus-aware)
- Concept explanations
- Smart summaries
- Flashcard generation
- Quiz/Test creation
- Revision guidance

### 🌐 DeepExplore Mode
- Concept breakdown
- Related topics
- Historical evolution
- Common confusions
- Debates & limitations
- Practical applications

### 🎯 Exam Strategy Intelligence
- PYQ pattern analysis
- Topic frequency detection
- Marks weightage insights
- High-priority unit suggestions

## 🛠️ Tech Stack

- **Frontend:** Next.js 14, React, TypeScript, Tailwind CSS
- **AI:** OpenAI GPT, Claude, Gemini
- **RAG:** Retrieval Augmented Generation
- **Database:** Supabase (PostgreSQL)
- **Vector Store:** Pinecone / Weaviate

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/NinadHirani/ai-academic-strategist.git
cd ai-academic-strategist
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env.local
# Add your API keys (OpenAI, Supabase, etc.)
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3001)
